# VertexFi

VertexFi is a decentralized finance (DeFi) protocol designed to provide secure, transparent, and permissionless financial services on OPN Testnet.

## Overview

VertexFi brings together core DeFi primitives into a single platform, enabling users to swap assets, provide liquidity, stake tokens, and participate in an open financial ecosystem.

## Features

- Token Swap
- Liquidity Pools
- Token Staking
- Yield Farming
- Vault Management
- Reward Distribution
- Governance Ready
- EVM Compatible
- Open Finance Infrastructure

## Technology

- Solidity
- Remix IDE
- MetaMask
- OPN Testnet

## Vision

To build an accessible and transparent DeFi ecosystem where anyone can securely exchange assets, earn rewards, and participate in decentralized finance.
